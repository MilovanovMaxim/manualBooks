#Setup
- `git clone git@github.com:MilovanovMaxim/manualBooks.git`
- `cd manualBooks`
- `npm install -g gulp`
- `npm install -g bower`
- `npm install`
- `bower install`

##Build
Run `gulp build` - for production

Run `gulp build --dev` - without minify

##For developer
Run `gulp serve`
